문제53: 이진 탐색 트리(BST) 노드 간 최소 거리

Created
Feb 12, 2023 03:13 PM
Tags
 

문제

두 노드 간 값의 차이가 가장 작은 노드의 값의 차이를 출력하라
 

입력

[4,2,6,1,3,null,null]
[10,3,15,1,8,null,null]
 

출력

1
2
 

풀이

  1. 재귀 구조로 중위 순회
class Solution(object):
    import sys
    prev = -sys.maxsize
    result = sys.maxsize
    def minDiffInBST(self, root):
        """
        :type root: TreeNode
        :rtype: int
        """
        
        def dfs(node):
            if node.left:
                dfs(node.left)
            
            self.result = min(self.result, node.val - self.prev)
            self.prev = node.val
            
            if node.right:
                dfs(node.right)

            return self.result
        
        return dfs(root)
